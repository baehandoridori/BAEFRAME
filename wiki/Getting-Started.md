## 설치

### 데스크톱 앱

**1. 저장소 클론**

```bash
git clone https://github.com/baehandoridori/BAEFRAME.git
cd BAEFRAME
```

**2. 의존성 설치**

```bash
npm install
```

**3. mpv 설치**

[mpv/README.md](https://github.com/baehandoridori/BAEFRAME/blob/main/mpv/README.md) 참고

**4. 실행**

```bash
npm start
```

### 웹 뷰어

설치 없이 바로 사용:

1. [baeframe.vercel.app](https://baeframe.vercel.app) 접속
2. 영상 URL 입력
3. 리뷰 시작

## 첫 실행

### 사용자 이름 설정

첫 실행 시 이름을 입력합니다. 댓글에 이 이름이 표시됩니다.

```
┌─────────────────────────────────┐
│  사용자 이름 설정               │
├─────────────────────────────────┤
│                                 │
│  이름: [________________]       │
│                                 │
│  Slack 이름과 동일하게 입력     │
│                                 │
│            [확인]               │
└─────────────────────────────────┘
```

### 영상 열기

**방법 1**: 드래그 & 드롭

**방법 2**: `Ctrl + O` → 파일 선택

**방법 3**: `baeframe://` 링크 클릭

## 기본 사용법

### 영상 재생

| 동작 | 방법 |
|------|------|
| 재생/일시정지 | `Space` 또는 ▶ 버튼 |
| 1프레임 이동 | `←` `→` 또는 `,` `.` |
| 10프레임 이동 | `Shift + ←` `Shift + →` |
| 처음/끝으로 | `Home` / `End` |

### 댓글 달기

1. 원하는 프레임에서 일시정지
2. `C` 키 또는 💬 버튼 클릭
3. 댓글 입력 → `Enter`로 저장

```
Shift + Enter = 줄바꿈
Ctrl + B = 굵게
Ctrl + I = 기울임
```

### 그리기

1. `D` 키로 그리기 모드 진입
2. 도구 선택 (펜/화살표/지우개)
3. 색상, 굵기 선택
4. 영상 위에 그리기
5. 자동 저장됨

### 링크 공유

1. `Ctrl + Shift + C` 또는 "링크 복사" 버튼
2. Slack에 붙여넣기
3. 팀원이 클릭하면 해당 영상 자동 로드

## 단축키

### 재생

| 키 | 기능 |
|:--:|------|
| `Space` | 재생/일시정지 |
| `←` `→` | 1프레임 이동 |
| `,` `.` | 1프레임 이동 (대안) |
| `Shift + ←` `→` | 10프레임 이동 |
| `Home` | 처음으로 |
| `End` | 끝으로 |

### 구간 반복

| 키 | 기능 |
|:--:|------|
| `I` | 시작점 설정 |
| `O` | 종료점 설정 |
| `L` | 구간 반복 토글 |

### 그리기 / 댓글

| 키 | 기능 |
|:--:|------|
| `D` | 그리기 모드 |
| `Z` | 펜 도구 |
| `X` | 지우개 |
| `C` | 댓글 모드 |
| `Ctrl + Z` | 실행 취소 |
| `Ctrl + Y` | 다시 실행 |

### 타임라인

| 키 | 기능 |
|:--:|------|
| `Ctrl + 휠` | 확대/축소 |
| `Shift + 휠` | 가로 스크롤 |
| `\` | 전체 보기 |

### 레이어

| 키 | 기능 |
|:--:|------|
| `1` | 레이어 추가 |
| `2` | 레이어 삭제 |
| `3` | 키프레임 추가 |
| `Shift + 3` | 키프레임 삭제 |
| `4` | 프레임 삽입 |
| `5` | 프레임 삭제 |

### 파일

| 키 | 기능 |
|:--:|------|
| `Ctrl + O` | 파일 열기 |
| `Ctrl + S` | 저장 |
| `Ctrl + Shift + C` | 링크 복사 |
