## 영상 재생

### 지원 포맷

| 포맷 | 확장자 | 비고 |
|------|--------|------|
| H.264 | `.mp4` | 가장 일반적 |
| H.265 (HEVC) | `.mp4`, `.mov` | 고압축 |
| PNG 시퀀스 | `.mov` | 애니메이션 작업물 |
| ProRes | `.mov` | 고품질 |
| WebM | `.webm` | 웹 최적화 |

> **mpv 플레이어**를 사용하므로 대부분의 코덱을 지원합니다.

### 프레임 단위 탐색

애니메이션 리뷰의 핵심! 정확한 프레임 단위로 이동할 수 있습니다.

| 동작 | 단축키 | 설명 |
|------|:------:|------|
| 1프레임 뒤로 | `←` 또는 `,` | 24fps 기준 약 0.04초 |
| 1프레임 앞으로 | `→` 또는 `.` | 정밀한 프레임 탐색 |
| 10프레임 뒤로 | `Shift + ←` | 빠른 탐색 |
| 10프레임 앞으로 | `Shift + →` | 빠른 탐색 |

## 타임라인 마커 & 댓글

### 마커 시스템

영상의 특정 프레임에 마커를 찍고 댓글을 남깁니다.

```
00:00                                               05:30
├────────────────────────────────────────────────────┤
│      ●       ●              ●    ●                │
│    00:05:12  00:12:30       00:28:45  00:35:20    │
│      │         │               │        │         │
│   "손 위치"  "타이밍"       "궤적"    "표정"      │
```

### 댓글 작성

1. **댓글 모드 진입**: `C` 키 또는 💬 버튼
2. **댓글 입력**:
   - `Enter`: 저장
   - `Shift + Enter`: 줄바꿈
3. **리치 텍스트**:
   - `Ctrl + B`: **굵게**
   - `Ctrl + I`: *기울임*
   - `-` 입력: • 불릿 포인트

### 댓글 관리

- **해결됨 토글**: 완료된 피드백 표시
- **답글**: 스레드 형식 (더블클릭)
- **수정/삭제**: 본인 댓글만
- **필터**: 전체/미해결/해결됨
- **클릭 이동**: 해당 프레임으로 점프

## 화면 위 스케치

### 도구

| 도구 | 단축키 | 설명 |
|------|:------:|------|
| 펜 | `Z` | 자유 드로잉 |
| 화살표 | - | 방향 표시 |
| 지우개 | `X` | 그린 내용 삭제 |

### 옵션

- **색상**: 빨강, 노랑, 초록, 파랑, 흰색
- **굵기**: 1~5px
- **불투명도**: 조절 가능

### 프레임별 저장

그린 내용은 **현재 프레임에만** 표시됩니다.

```
Frame 1    Frame 2    Frame 3    Frame 4
  ○          ○          ●          ○
없음        없음      마킹 있음    없음
```

## 링크 공유

### baeframe:// 프로토콜

Slack에서 링크를 클릭하면 BAEFRAME이 자동 실행됩니다.

```
baeframe://G:/프로젝트/EP01/shot_015_v2.mp4
   │        └─────────────────────────────┘
 프로토콜              영상 파일 경로
```

### 링크 복사

- **단축키**: `Ctrl + Shift + C`
- **버튼**: "링크 복사"

---

## 버전 관리

### 자동 버전 감지

파일명에서 버전을 자동 파싱합니다.

```
shot_015_v1.mp4
shot_015_v2.mp4
shot_015_v3.mp4  ← 자동으로 그룹화
```

### 버전 전환

같은 폴더의 버전 파일들이 자동 감지되어 UI에서 선택할 수 있습니다.

## 사용자 설정

### 이름 설정

첫 실행 시 입력한 이름이 댓글에 표시됩니다.

### 특별 사용자

| 이름 | 색상 | 비고 |
|------|------|------|
| 배한솔/한솔 | 빨강 | 특별 아바타 |
| 윤성원 | 파랑 | - |
| 허혜원/모몽가 | 핑크 | 특별 아바타 |

### 자동 저장

모든 설정이 자동 저장됩니다:
- 사용자 이름
- UI 패널 크기
- 그리기 도구 설정

## .bframe 파일

리뷰 데이터는 `.bframe` 파일로 저장됩니다.

```
📁 프로젝트 폴더
├── EP01_shot_015_v2.mp4       ← 원본 영상
└── EP01_shot_015_v2.bframe    ← 리뷰 데이터
```

Google Drive에 저장하면 팀원들과 자동 동기화됩니다.
