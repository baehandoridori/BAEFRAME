# UI 및 단축키 개선 개발 계획

## 요약

| 항목 | 수정 파일 | 이유 | 우선순위 | 상태 |
|------|----------|------|---------|------|
| Phase 1 | user-settings.js | 드로잉 기본 단축키 D→B 변경 | 높음 | ✅ 완료 |
| Phase 2 | index.html, main.css | 시작화면/헤더 버튼 이모지→SVG | 높음 | ✅ 완료 |
| Phase 3 | app.js | B/E 키 브러시/지우개 토글 | 중간 | ✅ 완료 |
| Phase 4 | app.js, user-settings.js | Shift+화살표 10프레임, Ctrl+화살표 1초 | 높음 | ✅ 완료 |
| Phase 5 | user-settings.js, index.html, app.js, main.css | 프레임/초 이동값 설정 기능 | 중간 | ✅ 완료 |

## 배경 및 목적

사용자가 요청한 여러 UI/UX 개선 사항:
1. 드로잉 버튼이 'D'로 표기되어 있지만 실제로는 'B'가 드로잉 모드 진입 키
2. 시작화면 및 헤더 버튼들의 이모지를 세련된 SVG 아이콘으로 교체
3. 그리기 모드에서 B(브러시), E(지우개) 토글 기능 추가
4. Shift+화살표가 현재 댓글 이동인데 10프레임 이동으로 변경
5. Ctrl+화살표로 1초씩 이동하는 기능 추가
6. 이동 프레임/초 값을 설정에서 변경 가능하게

## 구현 완료 내역

### Phase 1: 드로잉 기본 단축키 D→B 변경 ✅

**수정 파일:** `renderer/scripts/modules/user-settings.js`

```javascript
// 변경 완료
drawMode: { key: 'KeyB', ctrl: false, shift: false, alt: false, label: '그리기 모드' },
```

### Phase 2: 시작화면/헤더 버튼 이모지→SVG ✅

**수정 파일:** `renderer/index.html`, `renderer/styles/main.css`

| 요소 | 이전 | 변경 후 |
|------|------|--------|
| 영상 열기 버튼 | 🎬 | SVG (재생 아이콘) |
| 파일을 열어주세요 텍스트 | 클릭 불가 | 클릭 가능한 버튼 + SVG |
| 폴더 열기 | 📂 | SVG (폴더 아이콘) |
| 다른 파일 | 🎬 | SVG (파일+재생 아이콘) |
| 저장 | 💾 | SVG (저장 아이콘) |
| 링크 복사 | 🔗 | SVG (링크 아이콘) |

**추가 CSS:**
- `.btn-large`: 큰 버튼 스타일 (시작화면용)
- `.file-name-clickable`: 클릭 가능한 파일명 스타일

### Phase 3: B/E 키 브러시/지우개 토글 ✅

**수정 파일:** `renderer/scripts/app.js`

- B키: 드로잉 모드 진입 + 브러시 도구 선택
- E키: 드로잉 모드 진입 + 지우개 도구 선택
- V키: 드로잉 모드 종료

### Phase 4: Shift+화살표 10프레임, Ctrl+화살표 1초 ✅

**수정 파일:** `renderer/scripts/app.js`, `renderer/scripts/modules/user-settings.js`

| 단축키 | 이전 동작 | 변경 후 |
|--------|----------|--------|
| Shift+← | 이전 댓글로 이동 | 설정된 프레임 수만큼 뒤로 (기본 10) |
| Shift+→ | 다음 댓글로 이동 | 설정된 프레임 수만큼 앞으로 (기본 10) |
| Ctrl+← | (없음) | 설정된 초만큼 뒤로 (기본 1초) |
| Ctrl+→ | (없음) | 설정된 초만큼 앞으로 (기본 1초) |

**추가 설정:**
- `prevSecond`, `nextSecond` 단축키 정의 추가

### Phase 5: 프레임/초 이동값 설정 기능 ✅

**수정 파일:**
- `renderer/scripts/modules/user-settings.js`: `frameSkipAmount`, `secondSkipAmount` 설정 추가
- `renderer/index.html`: 단축키 설정 모달에 이동량 설정 UI 추가
- `renderer/styles/main.css`: `.skip-settings-*` 스타일 추가
- `renderer/scripts/app.js`: 설정값 로드/저장 로직 추가

**설정 범위:**
- 프레임 이동량: 1 ~ 100 프레임
- 초 이동량: 0.1 ~ 10 초

## 테스트 및 확인 목록

- [ ] 드로잉 버튼에 'B'로 표시되는지 확인
- [ ] 단축키 메뉴(?)에서도 'B'로 표시되는지 확인
- [ ] 단축키 메뉴에 '지우개 모드 (E)' 표시 확인

### 2. SVG 아이콘 확인
- [ ] 시작화면 영상 열기 버튼 SVG 아이콘 표시
- [ ] 헤더 버튼들 (폴더열기, 다른파일, 저장, 링크복사) SVG 아이콘 표시

### 4. 프레임/초 이동

- [ ] 단축키 메뉴에 "1초 이동 (Ctrl+←→)" 표시
